 ⛯ Divine Craft ⛯

Example game built with @divinevoxel/voxel, @dvegames/vlox, and @babylonjs/core.

The textures used are from the resource pack Pixel Perfection by XSSheep.

## To Start

```console
git pull
git lfs pull
git submodule update --init --recursive
npm run init:all
npm install
npm run start
```
